
# Problem

Given a string of numbers and operators, return all possible results from
computing all the different possible ways to group numbers and operators. The
valid operators are `+`, `-` and `*`.

Input: `"2-1-1"`.

Output: `[0, 2]`

Input: `"2*3-4*5"`

Output: `[-34, -14, -10, -10, 10]`

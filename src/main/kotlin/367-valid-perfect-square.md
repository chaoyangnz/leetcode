
# Problem

Given a positive integer _num_, write a function which returns True if _num_
is a perfect square else False.

**Note:** **Do not** use any built-in library function such as `sqrt`. 

**Example 1:**
    ```
    Input: 16
    Returns: True
    ```

**Example 2:**
    ```
    Input: 14
    Returns: False
    ```

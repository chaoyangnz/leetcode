# Problem

Given an integer, write a function to determine if it is a power of two.

# Analysis

很显然，2的幂的二进制表示一定其中只有一个1，如：
```
1   - 00000000 00000001
2   - 00000000 00000010
4   - 00000000 00000100
8   - 00000000 00001000
16  - 00000000 00010000
32  - 00000000 00100000
64  - 00000000 01000000
128 - 00000000 10000000
...
```
